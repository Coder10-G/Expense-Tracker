# 💰 Expense Tracker (Python + Pandas)

A simple terminal-based Expense Tracker built using Python.  
This project helps users track expenses, store them in a CSV file, and analyze spending using category-wise summaries and a dashboard view.

---

## 🚀 Features

- Add new expenses (Name, Category, Amount)
- Store data in a CSV file
- Input validation (no empty or invalid data)
- Category-wise expense analysis
- Total expense calculation
- Top spending category detection
- Clean dashboard-style output

---

## 📊 Dashboard Output Includes

- Category-wise breakdown
- Total expenses
- Highest spending category
- Clean structured terminal UI

---

## 🛠️ Technologies Used

- Python 🐍  
- Pandas 📊  
- CSV file handling  
- Basic data analysis logic
